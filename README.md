> 言語・フレームワーク・ライブラリ
* Java 1.8
* Spring Boot 1.5.9 (Spring MVC 4.3.13)
* Spring Data JPA(Hibernate) 1.11.9
* Spring Security 4.2.3
* Bootstrap 3.3.7
* jQuery 1.11.1
* Thymeleaf 3.0.7
* lombok 1.16.18
* Spring boot devtools 1.5.9

> ビルドツール
* Gradle 4.5

> テストツール
* Junit 4.12

> APサーバ(Webサーバと同居)
* Apache Tomcat 8.5

> DBサーバ
* Oracle 11g
